## OldPart Precision

Single-page `Next.js` landing page for a heritage automotive precision-parts brand.

## Local Setup

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Run the verification commands:

```bash
pnpm vitest run
pnpm lint
pnpm build
```
